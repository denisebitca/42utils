utils

(unofficial)

a project by denise. this website connects to google via google fonts (which might be illegal in germany).

This is supposed to be a list of utilities for 42 students across campuses. Any ideas? Add a comment on Github. Send a message via 42utils@42l.fr to request takedowns.

Some utilities require a working 42 intranet account.

  • Slots-finder by Ludovic Lemaire
    • An app that sends a notification when a "correction slot" is available for your project.
  • Find Peers by Joppe Koers
    • A website that helps students to find peers that are working on the same project.
  • 42evaluators by rfautier (mainly for 42 Paris)
    • Taken from the website, changed a little: A website that solves a problem that all post-stage-students are facing: They can't find an evaluator for a project.
  • 42randoms by pcartau
    • Taken from the website, changed a little: A website for sharing resources that are useful to current and future students of 42 campuses (mainly 42 Paris for now).
  • Francinette by xicodomingues
    • An easy to use testing framework for 42 projects.
  • LazyPhilosophersTester by MichelleJiam
    • A semi-automatic tester for checking 42's philosophers project. It tests: invalid inputs, when program should stop on death/eaten enough, and when program should stay running (timed).
  • push_swap tester by Simon Cros
    • A fast push_swap tester (calculates complexity too!)
  • ftpkg_cli by Denise Bitca (42 Paris only)
    • Terminal utility to install packages curated by the 42 Paris staff on 42 Paris dumps.
  • MLX docs by harm-smits
    • A tutorial-style documentation for the MLX. (I'd also read the man file if I were you!)
  • MLX Starter Pack by pulgamecanica
    • Blog post about the best way to start a project that uses the MLX.
  • Raycasting tutorial by permadi
    • Blog post about raycasting, with examples.
